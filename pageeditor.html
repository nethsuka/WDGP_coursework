Home Page

Created the template, navigation bar, and home page. To ensure that every webpage has the same appearance, an external CSS style sheet was also created. Added the website's logo in the upper left corner. You can find the website mission by slightly scrolling through the homepage. Four cards were made with links to each student's content page. Added two buttons: one for the sitemap and the other for the feedback form. There is a page editor button in the footer. Once the user click on it from a particular page, it directs the user to the respective Page editor of that student who created that page.


Feedback Form

The feedback form page was designed with an easy-to-use interface that makes it simple for users to navigate and submit comments.
It consists of the following components:
1. Input fields
2. Validation
3. Error handling
4. Preview
5. Edit
6. Confirmation
Furthermore added required-fill input fields for users to provide their email address and name. Validated the email address using javascript. When users try to submit the form with incomplete or inaccurate information, they are met with clear error messages. Users can preview and edit before submitting, if necessary. Upon successful submission, a confirmation message will appear.


Content page
